Object Oriented Programming

Object-oriented programming (OOP) is a computer programming model that organizes software design around data, or objects, rather than functions and logic.

Class

Class is a collection of methods and objects that perform the set of instructions defined. It consists of variable which define the state of the class and behaviour which depends on the state of the class. The state can be changes by using the methods or functions. Method consists of the instruction which may update the state of the class

Inheritance

Inheritance is a way of creating a new class for using details of an existing class without modifying it. The newly formed class is a derived class (or child class) and the existing class is a base class (or parent class). There are 4 types of Inheritance as follows

class Calculations:
"""This is the base class"""
def __init__(self, list_passed_by_user):
self.list_1 = list_passed_by_user


class Addition(Calculations):
"""This is the child class"""
def get_output(self):
result = 0
for i in self.list_1:
result = Addition.add(i, result)
return result
Encapsulation

Using OOP in Python, we can restrict access to methods and variables. This prevents data from direct modification which is called encapsulation. Encapsulation is defined as the wrapping up of data under a single unit. In Python, we denote private attributes using underscore as the prefix i.e single _ or double __.
